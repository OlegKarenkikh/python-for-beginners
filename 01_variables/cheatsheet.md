# Шпаргалка — Глава 01: Переменные и типы

## Создание переменных
```python
name = "Иванов"       # str — строка
age = 35              # int — целое число
rate = 1.5            # float — дробное число
active = True         # bool — True или False
```

## Проверка типа
```python
type(name)     # <class 'str'>
type(age)      # <class 'int'>
```

## Вывод на экран
```python
print(name)               # Иванов
print(f"Имя: {name}")     # Имя: Иванов (f-строка)
print(f"{rate:.2f}")      # 1.50 (2 знака после запятой)
print(f"{age:,}")         # 35 (с разделителем тысяч)
```

## Имена переменных: правила
```python
# ✅ Правильно
client_name = "Иванов"
base_rate_2024 = 12000

# ❌ Неправильно
2rate = 1.5         # нельзя начинать с цифры
my rate = 1.5       # нельзя пробелы
```

## Арифметика
```python
a + b   # сложение
a - b   # вычитание
a * b   # умножение
a / b   # деление (результат float)
a // b  # целочисленное деление
a % b   # остаток от деления
a ** b  # возведение в степень
```
