## Что такое Python?

Python — это язык программирования. Программирование — это способ объяснить компьютеру что делать.

Представьте, что вы объясняете задачу новому сотруднику:
- Чем точнее объясните — тем точнее сделает
- Python — это язык, на котором говорит компьютер

**Почему Python, а не другой язык?**

| Язык | Для чего | Сложность для новичка |
|---|---|---|
| Python | Всё: автоматизация, данные, ИИ | ⭐ (легко) |
| Java | Корпоративные системы | ⭐⭐⭐ |
| C++ | Игры, системы | ⭐⭐⭐⭐⭐ |
| SQL | Базы данных | ⭐⭐ |

## Что вы сделаете в этом курсе

К концу курса вы напишете скрипт, который:
- Считает страховую премию для любого клиента
- Проверяет заявление на выплату по правилам
- Читает список клиентов из файла и обрабатывает каждого
- Сохраняет результат в JSON-файл

**Пример финального скрипта (вы его напишете сами):**

```python
# Финальный проект: калькулятор страховой премии
def calculate_premium(age, experience, accidents):
    base = 12_000
    if age < 25:
        base *= 1.5
    if accidents > 0:
        base *= (1 + accidents * 0.2)
    if experience > 10:
        base *= 0.85
    return round(base, 2)

clients = [
    {"name": "Иванов А.П.", "age": 35, "experience": 8, "accidents": 0},
    {"name": "Петрова М.С.", "age": 22, "experience": 2, "accidents": 1},
]

for client in clients:
    premium = calculate_premium(
        client["age"], client["experience"], client["accidents"]
    )
    print(f"{client['name']}: {premium:,.0f} руб./год")
```

```
Иванов А.П.: 10,200 руб./год
Петрова М.С.: 21,600 руб./год
```

Пока это выглядит непонятно — и это нормально! К главе 10 каждая строчка будет очевидна.
