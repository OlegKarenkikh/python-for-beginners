## Аналогия: числа — это рулетка страховщика

Страховой агент каждый день работает с числами:
- **Возраст** клиента (целое: 35, 22, 67)
- **Коэффициент риска** (дробное: 1.5, 0.85, 1.25)
- **Сумма выплат** (большое число: 1_500_000)

В Python есть два вида числовых «рулеток»:

---

## Типы чисел

```python
age     = 35          # int — целое число (без точки)
rate    = 1.15        # float — дробное (с точкой)
payout  = 1_500_000   # _ можно ставить как разделитель тысяч — удобнее читать
```

> 💡 **Правило:** используйте `int` для подсчёта штук, лет, количеств.
> Используйте `float` для коэффициентов и вычислений.

## Арифметика страхового калькулятора

```python
base          = 12_000      # базовая ставка ОСАГО
age_coeff     = 1.5         # молодой водитель
accident_coeff = 1.2        # одна авария

# Итоговая премия = base × коэффициент возраста × коэффициент аварий
premium = base * age_coeff * accident_coeff
print(f"Премия: {premium:,.2f} руб.")   # 21,600.00 руб.
#                         ↑ форматирование: запятая-разделитель, 2 знака после точки

# Полезные встроенные функции
print(round(21600.678, 0))    # 21601.0  — округлить до 0 знаков
print(abs(-5_000))            # 5000     — абсолютное значение (убирает минус)
print(max(1.2, 0.9, 1.5))    # 1.5      — максимальное из нескольких
print(min(1.2, 0.9, 1.5))    # 0.9      — минимальное
```

## Деление: важная деталь для новичков

```python
# / — всегда возвращает float (дробное)
10 / 2    # → 5.0  (не 5!)

# // — целочисленное деление (отбросить остаток)
47 // 5   # → 9   (47 дел на 5 агентов = по 9 каждому)

# % — остаток от деления
47 % 5    # → 2   (2 дела никому не досталось — осталось в очереди)

# Практический пример:
total_claims = 47
agents       = 5
per_agent    = total_claims // agents   # 9 дел на каждого
leftover     = total_claims % agents    # 2 дела в очереди
print(f"По {per_agent} дел на агента, ещё {leftover} ждут")
```

## Операторы сравнения и логика

```python
age        = 22
experience = 3
accidents  = 0

# Сравнение — результат True или False
young      = age < 25           # True  (22 < 25)
experienced = experience > 5    # False (3 > 5 = нет)

# and — оба условия должны быть True
# or  — хотя бы одно True
# not — инвертирует: not True = False
needs_surcharge = young and not experienced
print(needs_surcharge)    # True — молодой И неопытный = надбавка

# in — проверить, входит ли значение в список
high_risk_cities = {"Москва", "Санкт-Петербург"}
city = "Москва"
print(city in high_risk_cities)   # True
```
