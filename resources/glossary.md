# 📖 Глоссарий Python для начинающих

---

## Основные понятия

| Термин | Объяснение | Пример |
|---|---|---|
| **Переменная** | Ящик с именем для хранения данных | `age = 35` |
| **Тип данных** | Вид информации в переменной | `str`, `int`, `float`, `bool` |
| **Строка (str)** | Текст в кавычках | `"Иванов А.П."` |
| **Целое число (int)** | Число без дробной части | `35`, `0`, `-5` |
| **Дробное число (float)** | Число с точкой | `1.5`, `45000.0` |
| **Булево (bool)** | Истина или ложь | `True`, `False` |
| **Функция (def)** | Рецепт: написал один раз, используешь многократно | `def calculate(x): return x * 2` |
| **Список (list)** | Упорядоченный набор элементов | `[1, 2, 3]` |
| **Словарь (dict)** | Набор пар ключ:значение | `{"name": "Иванов", "age": 35}` |
| **Цикл for** | Перебор элементов по одному | `for item in list:` |
| **Цикл while** | Повтор пока условие выполняется | `while count < 10:` |
| **Условие if** | Развилка в коде | `if age < 25:` |
| **return** | Что функция возвращает в результате | `return premium` |
| **import** | Подключить готовый модуль | `import json` |
| **print()** | Вывести на экран | `print("Привет")` |
| **len()** | Длина строки или списка | `len("Иванов")` → 6 |
| **type()** | Узнать тип переменной | `type(35)` → `<class 'int'>` |
| **f-строка** | Строка с переменными внутри | `f"Имя: {name}"` |
| **индекс** | Позиция элемента (счёт с 0) | `list[0]` — первый элемент |
| **отступ** | 4 пробела после `if`/`for`/`def` | Обязателен! |
| **комментарий** | Пояснение в коде после `#` | `# это комментарий` |
| **JSON** | Формат хранения данных (словари в файле) | `{"key": "value"}` |
| **модуль** | Готовый файл с чужим кодом | `import statistics` |
| **pip** | Менеджер пакетов Python | `pip install requests` |
| **pytest** | Библиотека для автотестов | `def test_something(): assert ...` |
