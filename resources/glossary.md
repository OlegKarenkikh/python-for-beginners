# 📖 Глоссарий Python для начинающих

---

## Основные понятия

| Термин | Объяснение | Пример |
|---|---|---|
| **Переменная** | Ящик с именем для хранения данных | `age = 35` |
| **Тип данных** | Вид информации в переменной | `str`, `int`, `float`, `bool` |
| **Строка (str)** | Текст в кавычках | `"Иванов А.П."` |
| **Целое число (int)** | Число без дробной части | `35`, `0`, `-5` |
| **Дробное число (float)** | Число с точкой | `1.5`, `45000.0` |
| **Булево (bool)** | Истина или ложь | `True`, `False` |
| **Функция (def)** | Рецепт: написал один раз, используешь многократно | `def calculate(x): return x * 2` |
| **Список (list)** | Упорядоченный набор элементов | `[1, 2, 3]` |
| **Словарь (dict)** | Набор пар ключ:значение | `{"name": "Иванов", "age": 35}` |
| **Цикл for** | Перебор элементов по одному | `for item in list:` |
| **Цикл while** | Повтор пока условие выполняется | `while count < 10:` |
| **Условие if** | Развилка в коде | `if age < 25:` |
| **return** | Что функция возвращает в результате | `return premium` |
| **import** | Подключить готовый модуль | `import json` |
| **print()** | Вывести на экран | `print("Привет")` |
| **len()** | Длина строки или списка | `len("Иванов")` → 6 |
| **type()** | Узнать тип переменной | `type(35)` → `<class 'int'>` |
| **f-строка** | Строка с переменными внутри | `f"Имя: {name}"` |
| **индекс** | Позиция элемента (счёт с 0) | `list[0]` — первый элемент |
| **отступ** | 4 пробела после `if`/`for`/`def` | Обязателен! |
| **комментарий** | Пояснение в коде после `#` | `# это комментарий` |
| **JSON** | Формат хранения данных (словари в файле) | `{"key": "value"}` |
| **модуль** | Готовый файл с чужим кодом | `import statistics` |
| **pip** | Менеджер пакетов Python | `pip install requests` |
| **pytest** | Библиотека для автотестов | `def test_something(): assert ...` |

## Продвинутые понятия (главы 11–13)

| Термин | Объяснение | Пример |
|---|---|---|
| **FastAPI** | Фреймворк для создания HTTP API на Python | `@app.get("/clients")` |
| **эндпоинт** | Один URL-адрес в API | `GET /clients`, `POST /premium` |
| **HTTP-метод** | Тип запроса: GET (читать), POST (создать), DELETE (удалить) | `requests.get(url)` |
| **Pydantic** | Библиотека валидации данных. Проверяет типы и диапазоны | `class Client(BaseModel)` |
| **uvicorn** | ASGI-сервер для запуска FastAPI | `uvicorn main:app --reload` |
| **LLM** | Large Language Model — большая языковая модель | GPT-4, LLaMA, Gemma |
| **инференс** | Запуск обученной модели для получения ответа (не обучение) | `ollama run llama3.2` |
| **Ollama** | Программа для запуска LLM локально, на вашем компьютере | `ollama serve` |
| **vLLM** | Высокопроизводительный движок инференса для GPU-серверов | `vllm serve model` |
| **промпт** | Текстовый запрос к языковой модели | `"Ты страховой агент..."` |
| **temperature** | Параметр «творчества» модели: 0 = стабильно, 1 = случайно | `temperature=0` |
| **агент** | Программа: LLM + инструменты + цикл принятия решений | `while not done: ask_llm()` |
| **инструмент** | Обычная Python-функция, которую агент может вызвать | `def calculate_premium()` |
| **ReAct** | Паттерн агента: Reason (подумать) → Act (действовать) | `Thought: ... Action: ...` |
| **декоратор** | Функция-обёртка, изменяющая поведение другой функции | `@app.get("/")` |
| **JSON Schema** | Описание структуры JSON-объекта (типы, обязательность полей) | `{"type": "object", "properties": ...}` |
| **HITL** | Human-In-The-Loop — человек в цепочке решений агента | Кнопка «подтвердить» перед выплатой |
| **pipeline** | Цепочка шагов обработки данных | `read → validate → calculate → write` |
| **embedding** | Числовое представление текста для поиска схожих фрагментов | `[0.12, -0.34, 0.78, ...]` |
| **скрытые артефакты** | Неявные знания эксперта, не задокументированные в системе | Опыт Марины в оценке рисков |
| **виртуальное окружение** | Изолированная среда Python с отдельными пакетами | `python -m venv .venv` |
| **SKILL.md** | Файл с описанием роли, контекста и задач для AI-агента | `ROLE: ты страховой аналитик` |
