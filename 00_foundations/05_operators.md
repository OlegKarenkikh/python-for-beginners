## Арифметические операторы

| Оператор | Название | Пример | Результат |
|---|---|---|---|
| `+` | Сложение | `12_000 + 3_000` | `15000` |
| `-` | Вычитание | `45_000 - 4_500` | `40500` |
| `*` | Умножение | `12_000 * 1.5` | `18000.0` |
| `/` | Деление | `45_000 / 12` | `3750.0` (всегда float!) |
| `//` | Целочисленное деление | `45_000 // 12` | `3750` (int) |
| `%` | Остаток от деления | `365 % 30` | `5` |
| `**` | Возведение в степень | `2 ** 10` | `1024` |

```python
# Страховой пример
annual_premium = 45_000
monthly = annual_premium / 12          # 3750.0
monthly_int = annual_premium // 12     # 3750 (без дробной части)
remainder = annual_premium % 12        # 0 (делится ровно)

# Сложный процент: 12 000 × 1.05^3
future_value = 12_000 * (1.05 ** 3)   # 13891.5
```

## Операторы сравнения

Всегда возвращают `True` или `False`.

| Оператор | Смысл | Пример | Результат |
|---|---|---|---|
| `==` | равно | `age == 35` | `True` / `False` |
| `!=` | не равно | `city != "Москва"` | `True` / `False` |
| `>` | больше | `premium > 50_000` | `True` / `False` |
| `<` | меньше | `accidents < 3` | `True` / `False` |
| `>=` | больше или равно | `experience >= 5` | `True` / `False` |
| `<=` | меньше или равно | `age <= 80` | `True` / `False` |

```python
age = 22

print(age == 22)    # True
print(age == 35)    # False
print(age > 18)     # True
print(age >= 25)    # False
print(age != 30)    # True
```

⚠️ **`==` (сравнение) и `=` (присваивание) — разные вещи!**

## Логические операторы

| Оператор | Смысл | Когда True |
|---|---|---|
| `and` | И | оба условия True |
| `or` | ИЛИ | хотя бы одно True |
| `not` | НЕ | инвертирует True/False |

```python
age = 22
experience = 2
accidents = 0

# and — оба должны выполниться
young_and_new = age < 25 and experience < 3    # True

# or — хотя бы одно
risky = age < 22 or accidents > 2              # True

# not — инверсия
safe_driver = not (accidents > 0)              # True

# Комбинация
needs_surcharge = (age < 25 or experience < 2) and accidents > 0
```

## Операторы принадлежности

```python
risk_cities = ["Москва", "СПб", "Екатеринбург"]
city = "Москва"

print(city in risk_cities)       # True
print("Казань" in risk_cities)   # False
print("Казань" not in risk_cities) # True

# В строках тоже работает
email = "agent@polisplus.ru"
print("@" in email)               # True
print(email.endswith(".ru"))      # True
```

## Операторы присваивания

```python
total = 0
total = total + 5_000    # длинная запись

# Короткая форма (+=, -=, *=, /=)
total += 5_000    # то же самое
total -= 1_000    # total = total - 1000
total *= 1.1      # total = total * 1.1
count = 10
count //= 3       # count = count // 3  →  3
```

## Приоритет операторов (от высшего к низшему)

```
1.  **         (возведение в степень)
2.  +x, -x    (унарный плюс/минус)
3.  *, /, //, %
4.  +, -
5.  ==, !=, <, >, <=, >=, in, not in
6.  not
7.  and
8.  or
```

```python
# Без скобок — умножение раньше сложения
result = 12_000 + 3_000 * 1.5    # = 16500, не 22500!

# Со скобками — явно
result = (12_000 + 3_000) * 1.5  # = 22500
```

💡 **Совет:** используйте скобки если сомневаетесь — код станет понятнее.
