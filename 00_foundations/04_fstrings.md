## Что такое f-строка

f-строка — строка с буквой `f` перед кавычкой.
Внутри фигурных скобок `{}` пишем переменные и выражения.

```python
name = "Иванов А.П."
age = 35
premium = 45000.50

print(f"Клиент: {name}, возраст: {age}")
# Клиент: Иванов А.П., возраст: 35
```

---

## Форматирование чисел

```python
premium = 45000.5

print(f"{premium}")          # 45000.5       — как есть
print(f"{premium:.0f}")      # 45001         — 0 знаков после точки
print(f"{premium:.2f}")      # 45000.50      — 2 знака
print(f"{premium:,.0f}")     # 45,001        — разделитель тысяч
print(f"{premium:,.2f}")     # 45,000.50     — тысячи + 2 знака
print(f"{premium:>15,.0f}")  #         45,001 — выравнивание вправо
print(f"{premium:<15,.0f}")  # 45,001         — выравнивание влево
```

---

## Проценты

```python
rate = 0.038

print(f"{rate:.1%}")    # 3.8%
print(f"{rate:.2%}")    # 3.80%
print(f"{rate:.0%}")    # 4%
```

---

## Строки

```python
name = "Иванов А.П."

print(f"{name}")          # Иванов А.П.
print(f"{name.upper()}")  # ИВАНОВ А.П.
print(f"{name:<25}")      # Иванов А.П.        (25 символов, влево)
print(f"{name:>25}")      #           Иванов А.П. (вправо)
print(f"{name!r}")        # 'Иванов А.П.'      (с кавычками)
```

---

## Целые числа

```python
n = 42
big = 1_500_000

print(f"{n:d}")        # 42
print(f"{n:05d}")      # 00042     — ведущие нули до 5 символов
print(f"{n:03d}")      # 042
print(f"{big:,}")      # 1,500,000
```

---

## Выражения внутри f-строк

```python
age = 35
base = 12_000
items = ["Иванов", "Петрова"]

print(f"Молодой: {age < 25}")         # Молодой: False
print(f"Премия: {base * 1.5:,.0f}")   # Премия: 18,000
print(f"Клиентов: {len(items)}")      # Клиентов: 2
```

## Таблица форматов

| Формат | Значение | Результат |
|---|---|---|
| `{val}` | `45000.5` | `45000.5` |
| `{val:.2f}` | `45000.5` | `45000.50` |
| `{val:,.0f}` | `45000.5` | `45,001` |
| `{val:.1%}` | `0.038` | `3.8%` |
| `{val:05d}` | `42` | `00042` |
| `{val:>15}` | `"текст"` | `          текст` |
| `{val:<15}` | `"текст"` | `текст          ` |
